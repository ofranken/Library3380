# Welcome to Procure!

This is a repository for Phase 3 of the CMP_SC 3380 Database course at MU with Dr. Rao. This system was designed by Isabella Bickhaus and Olivia Franken as a joint submission for the database project.

**Made With...**
- JavaScript (.JS and .JSX)
- CSS
- React
- Node.js
- MySQL as the open sourced Database Management System (DBMS)

**Required Installations:**
- VSCode (if you don't already have this installed, maybe don't try cloning this repo)
- MySQL Workbench ver 8.0.43
- Node.js
- Github Desktop (or some version of Git that you know how to use)

Make sure you have all of this stuff ^ installed before proceeding.

**Replicating this Repo:**

1. Pull the repository onto your local machine and open the folder in VSCode.
2. Open a terminal, cd into the repository and type: "npm run install"
3. Once all of the dependencies are installed, open up MySQL Workbench.
4. File > Open SQL Script > Navigate to the SQL folder under frontend/src in this repo that you recently cloned.
5. Open and run "InitializeLibraryDB.sql"
6. Open and run "SyntheticBookData.sql"
7. Open and run "SyntheticClassData.sql"
8. Return to the project where you opened it in VSCode.
9. Open a terminal and cd into the backend folder.
10. Enter "npm run start" to run the backend.
11. Open another terminal and cd into the frontend folder.
12. Enter "npm run start" to run the frontend.
13. Enjoy!
